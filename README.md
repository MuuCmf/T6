## MuuCmf T6

MuuCmf T6采用最新ThinkPHP6+MySql+Redis开发，是一个将CMS和Framework两个概念完美结合的内容管理系统及多应用整合开发解决方案，提供了丰富的系统功能，开发人员只需要专注于业务系统的开发。

## 环境

* PHP 推荐PHP7.4版本
* MySQL >= 5.6 且 <= 8.0 (需支持innodb引擎，目前版本8.0需取消严格模式)
* Redis >= 6
* Apache 或 Nginx
